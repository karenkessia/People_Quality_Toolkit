# Checklist de Qualidade — Onboarding

## Antes do Primeiro Dia
- [x] Papel e responsabilidades claramente definidos
- [x] Acesso a ferramentas provisionado
- [ ] Expectativas alinhadas com gestor(a)
- [x] Agenda da primeira semana definida

⚠️ Risco identificado: ausência de alinhamento prévio com gestor.

---

## Primeira Semana
- [x] Objetivo do papel compreendido pela pessoa
- [x] Contatos-chave apresentados
- [ ] Documentação acessível
- [x] Espaço seguro para perguntas

⚠️ Fricção: documentação técnica incompleta.

---

## Primeiros 30 Dias
- [x] Feedback inicial realizado
- [ ] Autonomia progressiva concedida
- [x] Expectativas revisitadas
- [ ] Fricções identificadas e registradas

---

## Riscos Identificados
- Dependência excessiva do time para tarefas simples.
- Insegurança sobre prioridades.

## Ações Preventivas
- Criar guia rápido de ferramentas.
- Definir tarefas de autonomia progressiva.
